# AI Horoscope

매일 아침 AI 모델들의 운세를 전하는 디지털 무당 제피로스

[오늘의 운세 보기](https://jiunbae.github.io/ai-horoscope/) | [@dev_horoscope](https://x.com/dev_horoscope)

## How it works

1. Every morning at 08:00 KST, GitHub Actions generates today's AI horoscope
2. Uses Google Gemini to create scenario-based recommendations
3. Model list is fetched from OpenRouter for accuracy
4. Tweet is posted to @dev_horoscope
5. Static site is rebuilt and deployed to GitHub Pages

## Project structure

```
ai-horoscope/
├── index.html              # Landing page (static, no build system)
├── assets/
│   └── logo.svg            # SVG logo
├── data/
│   ├── latest.json         # Today's horoscope (generated by Actions)
│   └── history.json        # Past horoscopes
├── scripts/
│   ├── generate.py         # Horoscope generator (Gemini + OpenRouter)
│   └── post_tweet.py       # Tweet poster (Tweepy)
├── prompts/
│   └── persona.md          # LLM persona for generation
├── .github/
│   └── workflows/
│       └── daily-horoscope.yml
└── README.md
```

## Required secrets

| Secret | Description |
|--------|-------------|
| `GEMINI_API_KEY` | Google Gemini API key (comma-separated for round-robin) |
| `TWITTER_API_KEY` | Twitter/X API key |
| `TWITTER_API_SECRET` | Twitter/X API secret |
| `TWITTER_ACCESS_TOKEN` | Twitter/X access token |
| `TWITTER_ACCESS_TOKEN_SECRET` | Twitter/X access token secret |

## Local development

```bash
# Generate horoscope locally
export GEMINI_API_KEY="your-key"
python scripts/generate.py

# Serve the static site
python -m http.server 8000
# Open http://localhost:8000
```
